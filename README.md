# AI Menstrual Symptom Analyzer

An AI-powered web application that helps users analyze menstrual symptoms and generate personalized insights, along with a downloadable PDF report.

---

## **Features**
- Input menstrual symptoms through a simple and user-friendly web form.
- AI-powered symptom analysis using the Groq API.
- Generate a PDF report summarizing the analysis.
- Easy-to-use interface built with HTML and CSS.
- Lightweight, secure, and ready for deployment.

---

## **Tech Stack**
**Backend:** Python, Flask  
**Libraries:** python-dotenv, os, FPDF  
**Frontend:** HTML, CSS  
**Environment Config:** `.env` for storing API keys securely
